This folder holds optional maintainer-only material (for example step-by-step
hosting notes). It is not required to run, test, or understand the application.

If you publish this repository publicly and prefer not to ship these files,
delete this entire folder before pushing, or add "maintainer/" to .gitignore
and stop tracking it (git rm -r --cached maintainer).

The repo root still contains railway.toml and Dockerfile when you deploy from
Git; see maintainer/RAILWAY.md only if you use Railway and want the full checklist.
